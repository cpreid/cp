# cp logger
simple logging lib for browser 


## Usage
```javascript
var logger = CPLogger({LOG_LEVEL: 'debug', height: '150px', open: false});
logger.error('This is an error');
logger.info('This is an info');
logger.debug('This is an debug NEW ONE');  
```
